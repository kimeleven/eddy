1. 프리즘라이브 서비스 분석

1.1 서비스 개요

프리즘 라이브 스튜디오(PRISM Live Studio)는 네이버(NAVER)가 2017년 출시한 라이브 스트리밍 소프트웨어입니다. OBS Studio 엔진 기반으로 PC(Windows/macOS)와 모바일(iOS/Android) 모두 지원합니다.

2. 핵심 기능 상세 분석

2.1 AR 필터 / 뷰티 필터 (PRISM Lens)

PRISM Lens는 독립 앱 형태의 카메라 필터 모듈입니다.

뷰티 필터

  • 5가지 보정 요소: Shape(얼굴형), Smooth(피부결), Eye(눈), Blush(피부톤), Lipstick(립)
  • 7가지 프리셋: K-pop, Glam, Soft, Bold, Natural, Cute, Sharp
  • 실시간 스트리밍 중 적용 가능

AR / 마스크 이펙트

  • 얼굴 인식 AR 카메라 효과 (마스크, 파티클)
  • Face Mosaic: 최대 7명 동시 모자이크
  • Atmosphere Filters: 레트로~현대 분위기 필터
  • GIPHY 연동 스티커 (매주 업데이트)
  • Reaction Touch Filters

2.2 캐릭터 / 아바타 변환 (VTuber)

핵심 차별점: 모바일에서 3D VRM 형식을 최초 적용한 스트리밍 앱
아바타 유형설명
PNG Tuber (2D)2장+ 이미지/GIF로 2D 캐릭터 생성
3D VRM VTuberVRM 형식 3D 모델, 실시간 표정 반영
VRoidHub 연동VRM 배포 플랫폼에서 모델 임포트

페이스 트래킹

  • 전면 카메라로 실시간 표정 캡처 → 아바타 반영
  • iOS: TrueDepth Camera (Face ID) 필수
  • 감정 이펙트: Joy, Happy, Angry, Sorrow, Fun

전신 모션 캡처 (Sony mocopi 연동)

  • Sony mocopi 기기로 전신 모션 데이터 전송
  • 페이스 트래킹 + 전신 모션 동시 결합
  • 스마트폰 단독으로 풀바디 VTuber 방송

8.3 개발 로드맵

Phase 1: 핵심 엔진 (4~6주)
  • MediaPipe Face Mesh 연동 — 실시간 얼굴 랜드마크 추출
  • 기본 뷰티 필터 구현 (스킨 스무딩, 페이스 리쉐이핑)
  • 카메라 입력 → AR 필터 적용 → 프리뷰 파이프라인
  • 플랫폼: 모바일(Flutter) + PC(Electron/Qt)
Phase 2: 아바타 시스템 (4~6주)
  • VRM 3D 모델 로딩 및 렌더링 (Three.js 또는 Unity)
  • MediaPipe 트래킹 데이터 → 아바타 파라미터 매핑
  • Live2D SDK 연동 (2D 아바타 옵션)
  • 표정/감정 이펙트 시스템
Phase 3: 스트리밍 (3~4주)
  • FFmpeg 기반 실시간 인코딩 (H.264, GPU 가속)
  • RTMP 출력 → YouTube/Twitch 동시 송출
  • 씬 관리 + 소스 전환 UI
  • 오디오 믹서
Phase 4: 고도화 (4~6주)
  • Banuba SDK 전환 (프리미엄 AR 필터 1,000+)
  • 멀티플랫폼 동시 송출 (최대 6채널)
  • PC ↔ 모바일 리모트 컨트롤
  • 라이브커머스 연동 (상품 오버레이)
Phase 5: 모션캡처 + 확장 (4주+)
  • MediaPipe Holistic → 상체 + 손 트래킹
  • 외부 모션캡처 기기 연동 (mocopi 등)
  • 커스텀 필터 에디터 (크리에이터용)
  • B2B SDK 패키징
